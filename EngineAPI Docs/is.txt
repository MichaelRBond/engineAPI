Is Functions

The following functions replace, or compliment, the built in PHP functions

is_function() : determines if passed in string is a function or method

is_odd() : determines if the integer passed in is Odd (not even)

isint() : determines if the passed in parameter is an integer

isnull() : Determines if the passed in parameter is NULL. Differs from built in is_null() in the following ways.
	1) if the string is the word "NULL" (upper or lower case) returns TRUE
	2) does a strict === NULL on the variable
	
isempty(), is_empty() : determines if the parameter passed in is empty. Differs from built in empty() in the following ways.
	1) Returns FALSE if the parameter is "0" or 0
	
isCLI : determines if engine is running from the command line
